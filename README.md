📱 使用方法

上传书籍

点击或拖拽上传 TXT 文件
等待加载完成


阅读和批注

点击任意段落添加批注
输入你的想法
点击"发送 🐾"保存


调整字号

点击 A− 减小字号
点击 A+ 增大字号


翻页

点击"上一页" / "下一页"
或使用键盘左右箭头（电脑上）



💾 数据存储

所有数据（书籍内容、批注、阅读进度）保存在浏览器本地
不需要登录，不需要网络
清除浏览器数据会丢失所有内容
建议定期截图保存重要批注
